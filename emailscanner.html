Email SMS Project

This project solved a real world problem for me. My plan was for this project to work as an extra layer of assurance that I would be made immediatly aware of any valuable emails that were coming in. I used Python to achieve this outcome.

This Project taught me about researching and implementing APIs into my scripts. For this project, I used the IMAP API to log into my email account and scrape my inbox for new emails and I used the Twilio API to send this information to my phone in the form of an SMS.
